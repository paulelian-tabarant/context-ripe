# Superpowers Iteration Documentation

This directory contains **iteration-local specifications and progress tracking** used during development
sprints and implementation cycles.

## Purpose

- **`specs/`** — iteration-specific notes, sketches, and working documents created during active development
- **`plans/`** (if present) — implementation plans for the current sprint

These are working documents and may be incomplete, redundant, or superseded as work progresses.

## For Project-Level Documentation

See the parent directories:

- **`docs/spec/index.md`** — complete, versioned design specification
- **`docs/architecture/decisions/ADR-###.md`** — individual architecture decision records
- **`docs/architecture/architecture.c4`** — C4 architecture diagrams

## Cleanup

Old iteration documents can be archived or deleted once the work is complete and findings have been
captured in the spec and ADRs.
